
{{alias}}( x )
    Computes the sine of an angle measured in degrees.

    Parameters
    ----------
    x: number
        Input value (in degrees).

    Returns
    -------
    y: number
        Sine.

    Examples
    --------
    > var y = {{alias}}( 0.0 )
    0.0
    > y = {{alias}}( 90.0 )
    1.0
    > y = {{alias}}( 30.0 )
    ~0.5
    > y = {{alias}}( NaN )
    NaN


{{alias}}( z )
    Rounds a double-precision complex floating-point number toward positive
    infinity.

    Parameters
    ----------
    z: Complex
        Complex number.

    Returns
    -------
    out: Complex
        Result.

    Examples
    --------
    > var v = {{alias}}( new {{alias:@stdlib/complex/float64}}( -1.5, 2.5 ) )
    <Complex128>
    > var re = {{alias:@stdlib/complex/real}}( v )
    -1.0
    > var im = {{alias:@stdlib/complex/imag}}( v )
    3.0
